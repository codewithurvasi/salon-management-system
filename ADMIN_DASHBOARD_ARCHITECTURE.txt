═══════════════════════════════════════════════════════════════════════════════
                            DATA FLOW EXAMPLE
═══════════════════════════════════════════════════════════════════════════════

1. USER INTERACTS WITH UI
   └─> Clicks filter dropdown or sorts table


2. COMPONENT HANDLES EVENT
   └─> FilterBar's onChange triggers updateFilter()


3. HOOK UPDATES STATE
   └─> useFilters() updates filters object


4. COMPONENT RE-RENDERS
   └─> DataTable receives new filtered data


5. HELPER FUNCTION FORMATS DATA
   └─> formatDate(), getStatusBadgeColor() etc.


6. DISPLAY UPDATES
   └─> User sees new filtered/sorted results


Example Flow:
─────────────

User clicks: "Filter by Confirmed"
    ↓
FilterBar.onChange triggers: updateFilter('status', 'Confirmed')
    ↓
useFilters hook updates state: filters = { status: 'Confirmed' }
    ↓
Component re-renders with filtered appointments
    ↓
Each appointment's status is colored via: getStatusBadgeColor()
    ↓
User sees "Confirmed" appointments highlighted in green
